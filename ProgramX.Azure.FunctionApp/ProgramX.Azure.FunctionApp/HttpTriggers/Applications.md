
# Applications

Applications are portions of the wider platform intended to perform specific functions or roles.

## Querying Applications

Applications are returned using the `GET api/v1/applications{/(applicationName)[/health]|?[containsText=(text)&][withinRoles=(comma-separated-role-names)]}` endpoint.
