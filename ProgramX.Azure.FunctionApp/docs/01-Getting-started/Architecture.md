# Architecture

The architecture of the project adopts the following guiding principles:

* The platform is a web application.
* The platform is designed to provide a platform for providing services across a number of diverse applicatins and purposes.
* The platform is designed to be extensible.
* The platform is cheap.
* The platform doesnot require maintenance.
* The platform is easy to use and develop for.

## Front-end

The front-end is hosted on Azure Static Web Apps and uses a pure-HTML approach, utilising Angular as a means of adding 
dynamic behaviour.

The Tailwind CSS framework is used to provide a consistent and responsive design across the platform.

## Back-end

The back-end is hosted on Azure Functions and uses a serverless approach, utilising Node.js as a means of adding 
dynamic behaviour.

## Database

The database is hosted on Azure Cosmos DB and uses a NoSQL approach, utilising MongoDB as a means of adding 
dynamic behaviour.
